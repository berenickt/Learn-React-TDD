# jest 명령어

```json
"scripts": {
  // **** git commit되기 이전 것만 테스트코드로 바라보기
  // commit된 것들은 테스트 안함
  "test": "jest --watch",
  // **** 프로젝트 내 모든 코드를 테스트할 떄
  "test": "jest --watchAll"
},
```
